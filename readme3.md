#  hdu2012     
### **Problem Description**     
** 对于表达式n^2+n+41，当n在（x,y）范围内取整数值时（包括x,y）(-39<=x<y<=50)，判定该表达式的值是否都为素数。**    
## input   
###**  输入数据有多组，每组占一行，由两个整数x，y组成，当x=0,y=0时，表示输入结束，该行不做处理。**     
##  output
###**   对于每个给定范围内的取值，如果表达式的值都为素数，则输出“OK”，否则输出“sorry”，每组输入占一行。**       
## Sample Input          
**   0 1 **      
**   0 0 **       
#  Sample Output    
**  OK  **
